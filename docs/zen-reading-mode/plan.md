# 禅模式阅读 Plan

## 实现方案

- 禅模式是阅读页的临时低干扰显示状态，不持久化到浏览器或后端。
- 普通阅读页在顶部导航右侧提供禅模式入口，`Z` 在非交互目标上切换禅模式，`Escape` 退出禅模式。
- 进入禅模式时关闭阅读设置、倍速菜单和播放模式菜单，请求浏览器全屏；全屏请求失败时保留页面内禅模式并显示短暂轻提示。
- 浏览器全屏被用户退出后，同步退出禅模式；刷新、返回书架或切换阅读材料后恢复普通阅读页。
- 禅模式隐藏顶部导航、阅读材料标题区、来源信息和完整播放器，只保留正文、右上角退出按钮和底部常驻状态条。
- 禅模式状态条只显示朗读状态、时间进度、必要错误和重试入口；不常驻显示倍速、播放模式、时间轴拖动、上一篇/下一篇或篇末继续提示。
- 进入禅模式时滚动回当前句并恢复自动跟随；后续用户手动滚动仍按现有规则暂停自动跟随。

## 关键决策

- 不新增后端 API、数据库字段或本地持久化 key。
- 不把禅模式纳入阅读偏好、播放模式或阅读进度。
- 禅模式尊重现有阅读偏好主题，不强制深色或新增独立主题。
- 保留现有 `Space` 播放/暂停、`ArrowLeft` 快退 15 秒和 `ArrowRight` 前进 30 秒语义。
- 不新增 ADR；该功能是可回滚的前端显示状态，不涉及 schema 或不可逆兼容取舍。

## Topic 级验证

- 运行 `make check`。
- 浏览器验证：导航按钮进入、`Z` 进入/退出、`Escape` 退出、浏览器全屏退出同步、全屏失败降级提示、状态条错误/重试显示、当前句回滚定位、浅色/深色主题和窄屏布局。
- 回归验证普通阅读页播放器、阅读设置、播放模式、倍速、时间轴和 Space/左右方向键行为不变。
