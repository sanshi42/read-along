# Read Along

Read Along 把用户可访问的单篇来源内容转成忠实、可朗读、可断点续读的本地阅读材料。这个上下文关注阅读材料从导入到本地保存和阅读的完整生命周期。

## Language

**阅读材料（Reading Material）**:
材料库中的一篇可阅读内容，包含来源身份、标题、结构化正文、句子音频状态和阅读进度。
_Avoid_: 文章、文档、资源

**结构化正文（Structured Body）**:
从来源内容清洗得到、按顺序组织为段落和句子的忠实正文。它是正文等价性判断和 `content_hash` 的依据。
_Avoid_: 原始文本、提取文本

**材料库（Material Library）**:
保存阅读材料的本地私有集合，也指拥有阅读材料身份、重复判断、原子持久化、完整读取和删除生命周期的领域概念。
_Avoid_: Repository、数据库、存储层

**朗读列表（Playback List）**:
阅读页连续朗读时使用的阅读材料序列。当前版本不显式存储播放队列，而是复用材料库的导入顺序作为隐含列表。
_Avoid_: 播放队列、书架最近活动顺序

**朗读引擎（TTS Backend）**:
把当前句文本合成为本地音频文件的可替换能力。它决定朗读模型、声音、输出格式和是否会连接本机外部服务或在线服务。
_Avoid_: 系统声音、播放服务、音频播放器

**朗读模型（TTS Model）**:
朗读引擎用于生成语音的具体模型或服务端模型名称。它属于朗读音频生成选择，不属于阅读材料正文、阅读进度或播放模式。
_Avoid_: 阅读模型、LLM、声音文件

**来源身份（Source Identity）**:
一篇阅读材料的一个导入来源，由来源类型和稳定来源键标识，并保留来源 URI 用于展示和回查。一篇阅读材料可以关联多个来源身份。
_Avoid_: 来源字段、链接记录

**当前句（Current Sentence）**:
用户在阅读材料中当前定位、准备继续阅读或正在朗读的句子。它可以由手动选择或连续朗读改变，不等同于最后完整播放的句子。
_Avoid_: 播放句、已读句

**阅读进度（Reading Progress）**:
一篇阅读材料中当前句、播放倍速偏好与朗读完成状态的可恢复状态，表示用户准备继续阅读或重新朗读的位置。
_Avoid_: 播放历史、已读进度

**朗读位置（Playback Position）**:
当前句在阅读材料全部句子中的相对位置，用于提示从哪里恢复朗读，不表示用户已阅读的比例或完成度。
_Avoid_: 阅读完成度、已读比例

**阅读偏好（Reading Preferences）**:
用户跨阅读材料沿用的阅读外观选择，包括字号、行距与主题，不属于单篇阅读材料的阅读进度。
_Avoid_: 阅读设置、材料设置

**禅模式（Zen Mode）**:
阅读页中临时启用的低干扰阅读状态，用于隐藏非必要界面并让用户专注于正文和朗读状态。它不跨阅读材料保存，也不属于阅读偏好、播放模式或阅读进度。
_Avoid_: 全屏偏好、阅读偏好、播放模式

**播放模式（Playback Mode）**:
用户在当前浏览器中选择的朗读续播规则，决定自然篇末和跨阅读材料按钮如何使用朗读列表，不写入后端阅读进度。
_Avoid_: 阅读进度字段、材料属性、后端队列模式

**朗读会话（Playback Session）**:
用户打开一篇阅读材料后形成的临时朗读状态，包含当前句、句内位置、播放状态、播放倍速、播放模式和本轮音频准备结果。它随阅读页离开而结束，阅读进度只保存其中可恢复的部分。
_Avoid_: 音频播放器、播放队列、阅读进度

**朗读完成（Playback Completed）**:
阅读材料的最后一句自然播放结束后形成的状态，表示本轮连续朗读已到末尾，不表示用户已完成阅读。
_Avoid_: 阅读完成、已读
