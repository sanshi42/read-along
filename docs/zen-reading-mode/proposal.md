# 禅模式阅读 Proposal

## Goal

为阅读页提供一个可快速进入和退出的禅模式，让用户在边听边读时隐藏非必要界面，并用稳定的键盘操作控制朗读位置和播放状态。

目标完成后，用户应能通过快捷键进入近似全屏的低干扰阅读状态，在不触碰鼠标的情况下暂停、继续和调整朗读位置。

## Boundary

### In

- 阅读页的禅模式开关、退出入口和快捷键入口。
- 进入禅模式时隐藏或弱化顶部导航、阅读偏好面板入口、材料来源上下文等非必要阅读界面。
- 请求浏览器全屏；当浏览器拒绝全屏时，仍保留页面内低干扰模式。
- 与现有空格播放/暂停、左右方向键快退/快进保持一致的键盘控制。
- 明确 `j`、`k` 在朗读进度控制中的语义，并避免与输入框、按钮、滑块等交互控件冲突。
- 禅模式下当前句、高亮、时间式进度、播放器状态和阅读进度保存的一致性。

### Out

- 新增笔记、划线、统计、目录或全文搜索能力。
- 重新设计书架、导入面板或材料库管理流程。
- 改变朗读音频生成、缓存、材料导航和阅读进度领域语义。
- 引入账号、同步、多端或云端播放能力。
