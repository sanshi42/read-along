# 时间式朗读进度与跨材料导航 Proposal

## Goal

在句子级朗读的基础上，提供更接近视频播放器的整体朗读控制，使用户可以按时间快退、快进，并在阅读材料之间顺畅切换。

目标完成后，播放器左右按钮应切换上一篇和下一篇阅读材料，句内定位由时间式进度、快退快进或后续明确的句内控制承接。

## Boundary

### In

- 阅读页播放器中的整体时间式朗读进度表达。
- 快退 15 秒、前进 30 秒等时间单位控制。
- 播放器左右按钮切换相邻阅读材料。
- 当前句、高亮和阅读进度与时间式控制之间的一致性。
- 仅支持上一版六表结构到新增时长字段的窄迁移，迁移前备份旧数据库。

### Out

- 朗读音频预加载、批量生成和连续播放卡顿治理。
- 材料库搜索、标签、目录或批量管理。
- 多端同步、账号系统和云端播放历史。
- 更旧历史数据库、半迁移数据库或任意异常 SQLite 的 best-effort 恢复。
