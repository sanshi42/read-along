# Read Along MVP Proposal

## Goal

构建一个 macOS 优先的个人本地 Web App，把用户可访问的单篇网页或文本型 PDF 转成忠实、可朗读、可断点续读的阅读材料。

MVP 完成后，用户可以导入材料，在舒适的阅读页中播放句子级朗读、查看当前句高亮，并在刷新或重启后继续上次位置。

## Boundary

### In

- 在 macOS 本机运行的个人 Web App。
- 导入单篇公开网页、用户已授权访问的得到单篇课程页和文本型 PDF。
- 清洗正文并按段落、句子组织为阅读材料。
- 使用本机 TTS 生成句子级音频。
- 支持播放、暂停、上一句、下一句、倍速和当前句高亮。
- 保存本地阅读进度和阅读设置。
- 已导入正文和已生成音频可离线使用。

### Out

- 批量课程抓取、自动翻页或目录遍历。
- 绕过登录或付费权限，保存账号密码、Cookie 或导出的浏览器凭据。
- 扫描版 PDF OCR。
- LLM 改写、总结、出题或其他高级学习功能。
- 笔记、划线、标签和全文搜索。
- 独立导入任务进度、材料删除和其他材料管理能力。
- 多端同步、账号系统、手机 App 和公网部署。
