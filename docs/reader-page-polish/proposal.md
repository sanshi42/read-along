# 阅读页最终打磨 Proposal

## Goal

根据已有阅读页 critique，把恢复当前位置、正文语义、移动端播放器密度和阅读上下文打磨到可发布质量。

目标完成后，阅读页应在桌面和移动端可靠展示当前句，正文不再被辅助技术读成大量按钮，用户滚动到正文中段时仍能确认材料与朗读位置，移动端播放器不遮挡主要阅读路径。

## Boundary

### In

- 阅读页当前句可见性判断、滚动目标和“回到当前句”显隐规则。
- 阅读正文句子的语义和键盘/点击交互表达。
- 阅读页 sticky nav 的材料标题与句子进度上下文。
- 阅读页标题层级和来源信息降噪。
- 移动端播放器与正文安全区样式。
- 相关前端单元测试和浏览器级验证。

### Out

- 新增全局快捷键、跳转到第 N 句或完整进度概览。
- 改变播放模式、跨材料导航或音频准备策略。
- 重做整体视觉系统或新增设计 token。
