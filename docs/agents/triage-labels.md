# Triage 标签

工程技能使用五种标准 triage 角色。本文件将这些角色映射到本仓库 issue tracker 中实际使用的标签。

| 标准角色 | 本仓库标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，agent 无需额外人工上下文即可处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某种角色时，使用表格中对应的本仓库标签。

如果仓库后续更改标签名称，请同步更新本表右侧的本仓库标签。
